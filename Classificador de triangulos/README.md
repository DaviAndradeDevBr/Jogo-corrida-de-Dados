# Classificador de Triângulos em Node.js

## Descrição do Projeto
Este repositório contém uma implementação em **Node.js** destinada à classificação de triângulos. O algoritmo recebe as medidas dos três lados como entrada e determina a classificação geométrica do triângulo correspondente, além de validar se as medidas fornecidas formam um triângulo válido de acordo com a desigualdade triangular.

## Funcionalidades
* **Validação:** Verifica se os lados informados formam um triângulo válido.

* **Classificação:**
    * **Equilátero:** Todos os lados possuem medidas iguais.
    * **Isósceles:** Dois lados possuem medidas iguais.
    * **Escaleno:** Todos os lados possuem medidas distintas.

## Pré-requisitos
Para a execução deste projeto, é necessário ter instalado o [Node.js](https://nodejs.org/) (versão LTS recomendada).

## Como Executar
1.  Clone este repositório:
    ```bash
    git clone [https://github.com/DaviAndrade22/Classificador-de-Triangulos.git](https://github.com/DaviAndrade22/Classificador-de-Triangulos.git)
    ```
2.  Acesse a pasta do projeto via terminal:
    ```bash
    cd Classificador-de-Triangulos
    ```
3.  Execute o script principal:
    ```bash
    node triangulo.js
    ```
